Install your Command Centre

01

Quick — download the file

Best for getting started today
  1. Click Download the plugin to save the file.
  2. Open it in the Claude desktop app.
  3. Review the skills and press Add.
  4. Test it: type “be my board of advisors” — you’re in.
Download the plugin
02

Ongoing — add our library

Auto-updates as we add new staff
  1. In Claude, open the plugin / marketplace settings.
  2. Add this as a marketplace source:
your-github-username/founder-command-centre
  1. Install AI Her Way Founder Command Centre from the library.
  2. You’ll get every future skill and update automatically.

New agents are added to this same library over time — install once, grow forever.

No one gets left behind. Can’t install? Every skill also comes as a copy-paste version you can drop straight into a Claude Project — it works exactly the same way.
